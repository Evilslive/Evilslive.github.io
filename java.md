## Java vs C#

#### extends
用來擴充

可以是類&類之間的繼承、或是介面與介面之間的擴充

#### implements
用來實例化

傾向於類與介面之間的連結與應用(1個類連接多個介面, 同C#), 反過來說也就是解耦, 實例與抽樣分離
